SOM CGPA Calculator Project

This project contains two versions:

1. Python Version (Console Based)
   - Located in python_version folder
   - Run using: python cgpa.py

2. Web Version (Browser Based)
   - Located in web_version folder
   - Open index.html in browser

Grading System:
O = 10
E = 9
A = 8
B = 7
C = 6
D = 5
F = 2

Formula:
CGPA = Sum(GradePoint × Credit) / Total Credits
